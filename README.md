🛒 Amazon Mini Project

A responsive Amazon homepage clone built using HTML, CSS, and JavaScript. This project recreates the user interface of Amazon with a clean layout, responsive design, and interactive components. It was developed to improve frontend development skills and gain hands-on experience with real-world website design.


🌐 Live Demo

 https://amishabillava25.github.io/Amazon-mini-project/


🚀 Features

- Amazon-inspired responsive homepage
- Responsive navigation bar
- Search bar with category selection
- Hero banner section
- Product category cards
- Shopping cart icon
- Footer with multiple sections
- Hover effects and smooth UI interactions
- Mobile-friendly responsive layout
  

 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Font Awesome
- Git & GitHub
- GitHub Pages (Deployment)

The Amazon Mini Project is a responsive e-commerce website built using HTML, CSS, and JavaScript.
